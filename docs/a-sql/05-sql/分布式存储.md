# 分布式存储

## 存储类型

存储的最终目的是存放数据

### 分类方式

存储有多种分类方式：

* 本体存储、外置存储
* 链接方式
  * DAS（直连式存储，通过数据线直接连接主机）
  * SAN（存储区域网络，通过专用网络访问，共享块设备）
  * NAS（网络附加存储，通过网络对外提供服务，共享目录，如：ftp、samba）
* 块存储、文件存储、对象存储

#### 块存储

块设备：一切以磁盘形式存在的存储都是块设备，即存储块。

1. 块存储强调的是裸磁盘。
2. 所有以磁盘形式直接提供给主机或服务器访问的存储都是块存储。
3. 块存储使用时，文件系统在客户端。

#### 文件存储

存储块和文件系统打包后，对外提供存储服务

```mermaid

电脑-->网络-->文件系统-->物理磁盘
```

文件存储和块存储的区别：

1. 文件存储的存储端多了一层文件系统，通过文件系统的调度后，再对外提供存储资源。
2. 文件存储易于共享。

#### 对象存储

是一种全新的存储架构，结合了块存储和文件存储的优点。

特点：速度快、易于共享、扩展性强。

对象存储：二层结构的文件系统。

### 分布式存储

分布式存储系统，是通过网络将数据分散存储在多台独立的设备上。

分布式存储系统的特性：可扩展性、低成本、易用。

分布式存储接口方式可以是文件存储、块设备、对象存储。
