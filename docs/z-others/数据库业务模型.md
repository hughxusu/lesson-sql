

# 数据库业务模型

## 常见概念

### UUID

UUID是通用唯一识别码，让分布式系统中所有元素都能有唯一识别信息，不需要中央控制端来做辨识信息指定。可以使用UUID来代替自动增长主键。不推荐使用。

```mermaid
flowchart LR
    时间信息 --- 时钟序列 --- 唯一机器识别号
```

## 权限管理

常见权限模型

| 缩写 | 名称               |
| ---- | ------------------ |
| ACL  | 访问控制列表       |
| RBAC | 基于角色的权限控制 |
| ABAC | 基于属性的权限控制 |
| PBAC | 基于策略的权限控制 |

### 电商数据库

#### 基本概念

### SPU

Standard product unit 标准产品单位，描述一个产品的各种特性

### SKU

Stock keeping unit 库存进出计量单位，SKU 是物理上不可分割的最小存货单元。

### 商品参数设计

设计商品参数表，将商品参数变为字段，通过商品参数表与商品的联系，确定商品有多少个参数。商品参数的值用json存储在商品参数表中。
